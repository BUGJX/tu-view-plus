# Quick Start

This section describes how to use Tu view Plus in your project.

## 1.Install

```bash
npm install tu-view-plus
```

## 2.Import

> main.js

```javascript
// Full Import
import TuView from 'tu-view-plus';
Vue.use(TuView);

// On-demand Import
import { TuButton } from 'tu-view-plus';
Vue.use(TuButton);
```
